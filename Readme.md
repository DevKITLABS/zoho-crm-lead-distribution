# Zoho CRM Lead Distribution Orchestrator

Advanced Deluge automation for intelligent Lead assignment and qualification in Zoho CRM.

## Features

- Dynamic lead assignment
- Territory based routing
- Round robin allocation
- Lead scoring engine
- Duplicate lead detection
- SLA monitoring
- Auto follow-up task creation
- Spam lead filtering
- Sales manager escalation
- Lead recycling automation

## Trigger

Zoho CRM Workflow:
- Module: Leads
- On Create/Edit

## File

- `lead_distribution_orchestrator.deluge`

## Use Case

When a Lead enters CRM from:
- Website forms
- Facebook Ads
- Google Ads
- Third-party vendors
- API integrations

the automation:
- Calculates lead quality score
- Detects duplicates
- Filters spam leads
- Assigns leads to correct sales reps
- Creates SLA follow-up tasks
- Escalates overdue leads
- Updates campaign analytics
- Recycles inactive leads
